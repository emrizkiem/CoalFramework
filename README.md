A Core App Library (Coal) that simplifies and enhances app development.

## Installation

### CocoaPods

CoalFramework is available through [CocoaPods](https://cocoapods.org). To install it, simply add the following line to your `Podfile`:

```ruby
pod 'CoalFramework'
```

Then run:

```bash
pod install
```
